Nvm Yarn

05 May 2019 - 张赫

Error: invalid option: –without-node

当你用nvm的时候，安装yarn通常会执行这个命令：

brew install yarn --without-node

但是
目前（2019-05-05）在Mac系统下出现这个错误

Error: invalid option: --without-node

解决办法直接解决这个就可以了：

brew install yarn --ignore-dependencies

官方的解决办法是先正常安装yarn，然后在用brew强制覆盖yarn所带的node环境。
